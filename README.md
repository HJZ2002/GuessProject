🎮 How to Play

Select a difficulty — Easy, Medium, or Hard
Click Start Game
Type a number into the input field and press Guess or hit Enter
Use feedback clues to narrow down your answer:

🔥 Burning hot — very close
☀️ Warm — getting there
❄️ Freezing cold — far off


Use a Hint if you're stuck (limited per difficulty)
Win before your chances run out!

✨ Features

Hot/cold radar — proximity feedback after every guess
Hints — reveals the number's parity and which half of the range it falls in
Progress bar — turns from green → yellow → red as chances deplete
Scoring — rewards fewer guesses with higher points
Session tracker — tracks total score and win rate across rounds
